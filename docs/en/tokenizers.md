# Tokenizers

Tokenizers (and BPE) supported by toiro:

- janome
- nagisa
- mecab-python3
- sudachipy
- spacy
- ginza
- kytea
- jumanpp (via pyknp)
- sentencepiece
- fugashi-ipadic
- fugashi-unidic
- tinysegmenter
- tiktoken

!!! note
    Some tools (KyTea, Juman++ etc.) **require system installation**.
